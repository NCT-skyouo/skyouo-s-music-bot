## 設置
我假設您會了創建機器人, 複製機器人 token 和使用 git...
- 1. 首先下載 [node.js](https://www.nodejs.org/)
- Ubuntu/Debian: 
```bash
sudo apt install nodejs -y
npm install n -g
n 12
```
- 2. git clone 該 專案
- 3. 使用 ``npm install``
- 4. 配置 config/config.json
- 5. 使用 npm start
- 6. 享受你的音樂機器人!
